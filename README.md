# BMW Sales Dashboard

## Project Overview
This project analyzes BMW sales performance from 2020 to 2024 using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization
- Dashboard Design

## Key Insights
- Total Sales Volume: 84.9M+
- Best Selling Model: 7 Series
- Best Performing Region: Europe
- Sales Trend Analysis
- Fuel Type Analysis
